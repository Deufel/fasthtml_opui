Card

 Card (*c, variant='text', cls=None, **kwargs)
*Generates a

for cards with Open Props UI styling.

Args: *c: Children (e.g., Hgroup, Div(cls=“content”), Div(cls=“actions”)). variant: Variant (‘text’, ‘outlined’, ‘tonal’, ‘elevated’). Defaults to ‘text’. cls: Additional classes to add to the card. **kwargs: Additional HTML attributes.

Returns: FastHTML Div component with appropriate classes and children.*

c = Card(
    Hgroup(
        P("Overline"),
        H2("Headline", cls="h3"),
        P("Subhead")
    ),
    Div("Explain more about the topic shown in the headline and subhead through supporting text.", cls="content"),
    Div(
        Button("Share"),
        Button("Learn more"),
        cls="actions"
    ),
    variant="outlined"
)

print(c)
show(c)

 FieldGroup (*c, direction='column', disabled=False, error=False,
